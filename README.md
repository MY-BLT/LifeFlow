## 📖 شرح ایده

لایف‌فلو یک پلتفرم تحلیل بهره‌وری مبتنی بر هوش مصنوعی است که به کاربران کمک می‌کند:

- **الگوهای پنهان رفتاری** خود را شناسایی کنند
- بفهمند **چرا اهدافشان محقق نمی‌شود**
- **توصیه‌های هوشمند و شخصی‌سازی‌شده** دریافت کنند
- **زمان‌های اتلاف‌شده** را کشف کنند

### تفاوت با ابزارهای موجود

ابزارهای فعلی مانند Todoist، Notion، و Toggl فقط **چه** را نشان می‌دهند:
- چه کارهایی انجام شد
- چقدر زمان صرف شد
- چه برنامه‌ای دارید

**لایف‌فلو** به **چرا** پاسخ می‌دهد:
- چرا به اهدافتان نرسیدید
- چرا بهره‌وری پایین است
- چه الگوهایی مانع پیشرفت هستند

## 🚀 نحوه اجرا

### پیش‌نیازها
- Node.js 18+
- npm یا yarn

### نصب و اجرا

```bash
# کلون کردن پروژه
git clone <repository-url>
cd lifeflow

# نصب وابستگی‌ها
npm install

# اجرای محیط توسعه
npm run dev

# ساخت نسخه تولید
npm run build

# پیش‌نمایش نسخه تولید
npm run preview
```
